<b-resizer>

A Bosonic element which may be used to add resizable behavior to an element.

Check the demo.

Install

This element is available as a npm package, see the Bosonic documentation for Bosonic transpiler usage.

$ npm install --save b-resizer

Otherwise, you can download and include in your HTML file the builded CSS & JS files, alongside with Bosonic polyfills, as described in the Getting Started guide.

Usage

<div class="container">
    <div class="container-content">
        Lorem ipsum dolor sit amet, ...
    </div>
    <b-resizer>
</div>

<b-resizer> resizes its parent element in a bidirectional way by default.

API

Attributes

  • direction: sets the resize direction (left, right, top or bottom).
