Welcome to Bosonic !

Bosonic is a set of tools that enable you to build and use reusable Web Components now !

Getting started

What are Web Components ?

Web Components are based on a set of specs that provide developers with a solid foundation for building UI components as HTML custom elements. By building more advanced HTML elements that the browsers propose, you will be able to build more complex applications without making your code a total mess, dealing simultaneously with high-level application concepts and low-level DOM manipulation.

Why Bosonic ?

We loved the Introduction to Web Components spec. What we wanted was a way to build components as this spec describes, shielding ourselves against potential spec changes, and supporting not-so-modern browsers like IE9. And so Bosonic was born !

How it works

Bosonic is sort of a transpiler that takes Web Components in HTML files and transforms them into JS and CSS code, runnable on modern browsers.

Bosonic uses some Polymer polyfills (Custom Elements, Mutation Observers and Weak Maps) and brings its own polyfills, including a powerful yet simple one for Shadow DOM.
